Prerequisites
Unity Editor: Version 2022.3 LTS or higher (recommended).

Git: To clone the repository.

Installation
Clone the repository:

Bash
git clone https://github.com/WateryFox/Blade-Fruit-3D.git
Open Unity Hub.

Click Add and select the cloned project folder.

Open the MainScene located in Assets/Scenes/.

How to Play
Click and Drag (or swipe on touchscreens) to create a blade trail.

Slice the fruits to earn points.

Avoid the bombs—hitting one will end your run!

Built With
Unity 3D - Game Engine

C# - Scripting Language

Visual Studio / VS Code - IDE

Project Structure
Assets/Scripts: All C# logic for fruit spawning and slicing.

Assets/Prefabs: Pre-configured fruit and bomb objects.

Assets/Materials: Textures and shaders for the 3D models.
